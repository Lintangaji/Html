Tentang Element / Tag <!DOCTYPE html>

Seluruh dokumen HTML harus di awali dengan deklarasi <!DOCTYPE>

Deklarasi tersebut bukan tag HTML. Ini adalah "informasi" untuk penggunaan type dokumen yang ingin di gunakan. karna HTML memiliki beberapa tipe dokumen sebelum nya seperti contoh HTML 4.01

Dalam HTML 5, deklarasinya sederhana:

  
    
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html Doctype</title> 
  </head> 
  
  <body> 
  
  <!-- Hello World --> 
  
  </body> 
  </html>
