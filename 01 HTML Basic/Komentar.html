Tentang Element / Tag <!--komentar-->>

Tag komentar digunakan untuk memasukkan komentar dalam code anda.Â Komentar tidak ditampilkan di browser.

Anda dapat menggunakan komentar untuk menjelaskan code Anda, yang dapat membantu Anda saat mengedit code di lain waktu agar tidak lupa. Ini sangat berguna jika Anda memiliki banyak kode.
Contoh Sederhana : 

  
    
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html Doctype</title> 
  </head> 
  
  <body> 
  
  <!-- Ini Tidak Akan Di Tampilkan --> 
  <!-- Ini Hanyalah Tag Komentar --> 
  
  </body> 
  </html>
