Definisi Dan Cara Penggunaan tag <head>

Berisi metadata / informasi dokumen Tag <head> adalah wadah untuk meta data, di tempatkan di antara tag <html> dan tag <body>
Metadata adalah data yang berkaitan tentang dokumen html dan metadata tidak di tampilkan di halaman depan web
Metadata biasanya menentukan judul dokumen, kumpulan karakter, gaya, skrip, dan informasi meta lainnya.
Berikut ini adalah tag - tag yang bisa di sisipkan ke dalam tag <head> :


  • <title>
    • (di perlukan di setiap dokumen HTML)
  • <style>
  • <base>
  • <link>
  • <meta>
  • <script>
  • <noscript>
