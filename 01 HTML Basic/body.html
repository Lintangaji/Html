Tentang Element / Tag <body>

Tag <body> mendefinisikan tubuh dokumen.
Tag <body> berisi semua isi dokumen HTML, seperti judul, paragraf, gambar, hyperlink, tabel, daftar, dll..

Contoh Penulisan

     
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html Body</title> 
  </head> 
  
  <body> 
  
  <!-- Ketik Disini --> 
  
  </body> 
  
  </html>

Catatan: Hanya ada satu <body> elemen dalam dokumen HTML.
