Tentang Element / Tag <br>

Tag <br> menyisipkan satu baris tunggal / baru. Tag <br> berguna untuk menulis puisi.
Tag <br> adalah tag kosong yang berarti bahwa ia tidak memiliki tag akhir atau tag penutup.

Contoh Penulisan

    
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html Br</title> 
  </head> 
  <body> 
  
  <p>Everything Will Be <br /> Okay</p> 
  
  </body> 
  </html>

Hasil Penulisan

Everything Will Be
Okay
