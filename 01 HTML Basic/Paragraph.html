Tentang Element / Tag <p>

Tag <p> mendefinisikan sebuah paragraf. Browser secara otomatis menambahkan satu baris kosong sebelum dan sesudah setiap <p> elemen.

Contoh Penulisan

  
    
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html Doctype</title> 
  </head>
  <body> 
  
  <p>Hello Wolrd.</p> 
  
  
  </body> 
  </html>

Tip: Gunakan CSS untuk mengatur gaya paragraf.
