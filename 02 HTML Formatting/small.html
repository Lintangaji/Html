HTML ruby tag | belajar <small> element

Deskripsi

Tag <small> mendefinisikan teks yang lebih kecil (seperti hak cipta / copyright dan sisi-komentar lain).

tips: kamu bisa menggunakan element <small> jika ingin membuat text kecil seperti copy right dll..

atau menggunakan tag lain dengan menggunakan style CSS seperti pada contoh ke dua.

Di Bawah Ini Merupakan Contoh Penulisan Dan Hasil Contoh Penulisan

Contoh 1

HTML <small> element tanpa CSS


<!DOCTYPE html> 
<html> 
<head> 
<meta charset="UTF-8"> 
<meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar small element</title> 
</head> 
<body> 
  
<p><small>© 2021 by Lintang Aji</small></p>
  
</body> 
</html>

Contoh 2

HTML <small> element menggunakan CSS


<!DOCTYPE html> 
<html> 
<head> 
<meta charset="UTF-8"> 
<meta name="viewport" content="width=device-width, initial-scale=1">
<style>

span.small {
      font-size: smaller;
}

</style>
<title>Belajar small element</title> 
</head> 
<body> 
  
<p><span class="small">© 2021 by Lintang Aji</span></p>
  
</body> 
</html>
