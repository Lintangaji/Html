Tentang Element / Tag <mark>

Tag <mark> merupakan tag yang bisa menandai sebuah text di sertai dengan light kuning, tag ini bisa di gunakan pada bagian - bagian text yang di rasa Penting / Memang Perlu di tandai.

Contoh Penulisan

  
    
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html Doctype</title> 
  </head> 
  <body> 
  
  <p> jangan lupa untuk <mark>membeli susu</mark> 
  hari ini </p> 
  
  </body> 
  </html>

Hasil Penulisan

Jangan lupa untuk membeli susu hari ini
