Tentang Element / Tag <del> & Tag <ins>

Tag <del> di gunakan untuk mencoret sebuah text
Dan tag <ins> di gunakan untuk membuat garis bawah pada sebuah text

Contoh Penulisan Tag <del>

  
    
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html Doctype</title> 
  </head> 
  <body> 
  
  <p> 
  
  aku suka <ins>kamu</ins>
  kamu suka <del>dia</del>
  
  </p>
  
  </body> 
  </html>

aku suka kamu
kamu suka dia
