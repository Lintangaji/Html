Tentang Element / Tag <bdo>

BDO adalah singkatan dari Bi-Directional Override. TagÂ <bdo> digunakan untuk mengesampingkan arah teks saat ini / mengubah arah text dari kiri ke kanan.

Contoh Penulisan

     
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html bdo</title> 
  </head> 
  <body> 
  
  <p>text ini di bungkus hanya menggunakan tag p</p> 
  
  <p><bdo dir="rtl">text ini di bungkus hanya menggunakan tag p dan bdo</bdo></p> 
  
  
  </body> 
  </html>
