HTML samp tag | belajar <samp> element

Deskripsi

Tag <samp> digunakan untuk menampilkan output dari program komputer. Konten di dalamnya ditampilkan dalam font monospace default browser.

Tag : ini tidak digunakan lagi. Namun, dimungkinkan untuk mencapai efek yang lebih kaya dengan menggunakan CSS.

Di Bawah Ini Merupakan Contoh Penulisan Dan Hasil Contoh Penulisan

Contoh 1

HTML <samp> element


<!DOCTYPE html> 
<html> 
<head> 
<meta charset="UTF-8"> 
<meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar samp element</title> 
</head> 
<body> 
  
 <p>Message From My Computer:</p>
 <p><samp>File Not Found<br>press F1 to continue</samp></p>
  
</body> 
</html>

Hasil Contoh Penulisan

Message from my computer :

File Not Found
press F1 to continue
