Tentang Element / Tag <abbr>

Tag <abbr> mendefinisikan singkatan atau akronim, seperti "HTML", "CSS", "Mr.", "Dr.", "secepat mungkin", "ATM".

Tip: Gunakan atribut global untuk menampilkan deskripsi singkatan / akronim saat Anda mengarahkan mouse ke elemen. contoh misal gunakan dengan menggunakan Atribut title dll..

Contoh Penulisan

  
 <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html Abbr</title> 
  </head> 
  <body> 
  
   <p>
   
   Pertama Kalian Harus Belajar <abbr title="Hypertext Markup Language">HTML</abbr> lalu kalian bisa melanjutkan ke <abbr title="Cascading Stylesheet">CSS</abbr> 
   
   </p> 
  
  </body> 
  </html>

Hasil Penulisan

Pertama Kalian Harus Belajar HTML Lalu kalian bisa melanjutkan ke CSS
