HTML template tag | belajar <template> element

Deskripsi

Gunakan <template> untuk menyimpan beberapa konten yang akan disembunyikan saat halaman dimuat. Gunakan JavaScript untuk menampilkannya:

Tag <template> digunakan sebagai wadah untuk menampung beberapa konten HTML tersembunyi dari pengguna ketika beban halaman.

Konten di dalam <template> dapat dirender dengan JavaScript.

Anda dapat menggunakan <template> tag jika Anda memiliki beberapa kode HTML yang ingin Anda gunakan berulang kali,.

    yang akan disembunyikan saat halaman dimuat. Gunakan JavaScript untuk menampilkannya:

    <template> 
    <h2>Flower</h2>

    </template> 
    
    <script> function showContent() { 
    var temp = document.getElementsByTagName("template")[0]; 
    var clon = temp.content.cloneNode(true); document.body.appendChild(clon); } 
    </script>
