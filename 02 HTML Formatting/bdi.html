Tentang Element / Tag <bdi>

Pada contoh di bawah ini, nama pengguna ditampilkan bersama dengan jumlah poin dalam sebuah kontes. Jika tag <bdi> tidak didukung di browser, nama pengguna bahasa Arab akan membingungkan teks. (algoritme dua arah akan meletakkan titik dua dan angka "90" di sebelah kata "pemain" dari pada di samping kata "poin ").

Contoh Penulisan

     
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html bdi</title> 
  </head> 
  <body> 
  
  <ul>
  
  <li>pemain <bdi>Michael</bdi>: 60 point</li> 
  
  <li>pemain <bdi>Axel</bdi>: 60 point</li> 
  
  <li>pemain <bdi>Dewilma</bdi>: 60 point</li> 
  
  <li>pemain <bdi>لينتغ</bdi>: 60 point</li> 
  
  <li>pemain تابا: 60 point</li> 

  </ul> 
  
  </body> 
  </html>

Hasil Penulisan

  • pemain Michel: 60 point
  • pemain Axel: 60 point
  • pemain لينتغ: 60 point
  • pemain تابا :50 point
