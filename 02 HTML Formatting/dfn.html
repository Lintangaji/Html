Tentang Element / Tag <dfn>

Tag <dfn> digunakan untuk mendefinisikan / mengartikan perihal sebuah istilah (term).

Sebuah istilah yang akan dijelaskan pengertiannya dapat ditulis didalam element <dfn>, yang mana <dfn> itu sendiri harus terletak didalam element lain yang manaunginya, diantaranya element <p>, <section> atau <dt>.

Penjelasan dari istilah ditulis secara langsung setelah <dfn> element dan terletak didalam element yang menaunginya tersebut.

Contoh Penulisan

Contoh 1

Contoh <dfn> element yang menjelaskan Definisi / arti / kegunaan dari HTML

  1. <p>
  2. <dfn>HTML<⁄dfn> adalah bahasa markup yang digunakan untuk memformat struktur halaman web.
  3. <⁄p>

Contoh 2

Contoh <dfn> element menggunakan Atribut Title

  1. <p>
  2. <dfn title="Hypertext Markup Language">HTML<⁄dfn> adalah bahasa markup yang digunakan untuk memformat struktur halaman web.
  3. <⁄p>

Contoh 3

Contoh <dfn> Dengan menggunakan Tag <Abbr> di dalamnya

  1. <p>
  2. <dfn>
  3. <Abbr title="Hypertext Markup Language">
  4. HTML
  5. <⁄Abbr>
  6. <⁄dfn>
    7. adalah bahasa markup yang digunakan untuk memformat struktur halaman web.
  7. <⁄p>

Contoh 4

Contoh Lengkap Tag <dfn> dengan atribut id
sehingga bisa digunakan untuk mereferensikan definisi yang merujuk pada id tersebut menggunakanan <a> element.

  1. <p>
  2. <dfn id="html-def">
  3. <Abbr title="Hypertext Markup Language">
  4. HTML
  5. <⁄Abbr>
  6. <⁄dfn>
    7. adalah bahasa markup yang digunakan untuk memformat struktur halaman web.
  7. <⁄p>
