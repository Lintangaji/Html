Tentang Element / Tag <dfn>

Tag <dfn> digunakan untuk mendefinisikan / mengartikan perihal sebuah istilah (term).

Sebuah istilah yang akan dijelaskan pengertiannya dapat ditulis didalam element <dfn>, yang mana <dfn> itu sendiri harus terletak didalam element lain yang manaunginya, diantaranya element <p>, <section> atau <dt>.

Penjelasan dari istilah ditulis secara langsung setelah <dfn> element dan terletak didalam element yang menaunginya tersebut.

Contoh 1

Contoh <dfn> element yang menjelaskan Definisi "HTML" 

  
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html dfn</title> 
  </head> 
  <body> 
  
  <p> 
  
  <dfn>HTML</dfn> adalah bahasa markup yang di gunakan untuk memformat halaman web. 
  
  </p> 
  
  </body> 
  </html>

Contoh 2

Contoh <dfn> element dengan atribut title

  
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html dfn</title> 
  </head> 
  <body> 
  
  <p> 
  
  <dfn title="Hypertext Markup Language">HTML</dfn> adalah bahasa markup yang di gunakan untuk memformat halaman web. 
  
  </p> 
  
  </body> 
  </html>

Contoh 3

Contoh <dfn> dengan <abbr> element di dalam nya

  <html>

  </head>

     HTML
    </abbr>
    </dfn> adalah bahasa markup yang di gunakan untuk memformat halaman web. 
  
  </p> 
  
  </body> 
  </html>

Contoh 4

Contoh <dfn> element menggunakan id, sehingga bisa digunakan untuk mereferensikan definisi yang merujuk pada id tersebut menggunakanan <a> element 

  
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html dfn</title> 
  </head> 
  <body> 
  
  <p> 
  
    <dfn id="definisi-html">
    <abbr title="Hypertext Markup Language">
     HTML
    </abbr>
    </dfn> adalah bahasa markup yang di gunakan untuk memformat halaman web. 
  
  </p> 
  
  </body> 
  </html>
