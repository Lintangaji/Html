Tentang Element / Tag <kbd>

Tag <kbd> merupakan tag yang bisa merubah font text default HTML menjadi text seperti keyboard pada komputer.

Contoh Penulisan

  
    
  <!DOCTYPE html> 
  <html> 
  <head> 
  <meta charset="UTF-8"> 
  <meta name="viewport" content="width=device-width, initial-scale=1"> 
  <title>Belajar Html kbd</title> 
  </head> 
  <body> 
  
  <p>
  
  Tekan <kbd>Ctrl</kbd> + <kbd>s</kbd> Untuk melakukan perubahan / menyimpan file. 
  
  </p>
  
  </body> 
  </html>

Hasil Penulisan

Tekan Ctrl + s Untuk Melakukan Perubahan / menyimpan file

Yang arti nya, menyuruh user untuk menekan ctrl + s pada keyboard komputer, jika halaman web / aplikasi bisa melakukan perubahan dengan hanya menggunakan shortcut keyboard.

Tips : Jangan gunakan Tag kbd jika kalian hanya ingin merubah text agar terlihat berbeda, gunakan CSS!
