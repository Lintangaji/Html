Tentang Html

HTML adalah singkatan dari ( Hypertext Markup Language ) HTML merupakan bahasa markup yang di dalamnya terdapat serangkaian markup.
Dan nantinya markup tersebut kita akan panggil sebagi tag yang berfungsi untuk memberi tahu web browser bagaimana sebuah Konten akan di tampilkan.
Contoh :
Jika Kita ingin menampilkan sebuah text, maka text tersebut harus kita bungkus terlebih dahulu menggunakan sebuah tag/markup, yang berfungsi untuk memberi tahu web browser text yang kita buat akan di tampilkan sebagai apa?
apakah text tersebut ingin kita tampilkan sebagai paragraph, heading, atau sebagai hyperlink.
maka demi kian begitu penting nya tag/markup pada sebuah dokumen html agar web browser mudah memahami apa yang sedang kita buat.
